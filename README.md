# PDF拆分器

审图系统上传专用 —— 把整册PDF拆成单页，按"序号\_图号\_图名"自动命名。

## 用法

1. **选PDF** → 点浏览选择图册文件
2. **填标签** → 在图号/图纸名称输入框里填你这份图的图框里写的叫什么（比如"图号"、"图纸编号"、"编号"都行，**自动忽略空格**，写"图号"就能匹配"图    号"）
3. **点分析** → 等几秒，表格会出来每一页的识别结果
4. **人肉过一遍** → **务必看一眼**，有问题的直接双击表格里的格子改
5. **点拆分！** → 选个输出目录，搞定

## 注意事项（建议使用前读一遍）

### 1. 序号顺序

程序按PDF页序生成序号（1， 2, 3…），**没有排序调整功能**。所以转换前的PDF页序一定要对，有不想要的扉页/空白页，先在PDF里删干净再拖进来。

### 2. 只认文字，不认线条

这个工具**没有内置OCR**，只对**内嵌文字**的PDF有效。CAD导PDF时请注意：

- 大部分专业的图纸没问题（建筑、景观、暖通等）
- **结构图纸**要特别留意——有些结构图导出后图名不是文字，而是**矢量线条拼出来的字形**，程序识别不了
- 如果不确定，分析完看一眼表格里的图名对不对就知道

### 3. 竖排图纸容易抓不准

识别原理挺粗暴的：先搜到你指定的标签文字（比如"图号"）作为锚点，然后上下左右找对应的值。

- **横排图纸**（大部分页面）→ 效果很好
- **竖排图纸**（总平面图转了方向的）→ 容易抓歪，**手动改一下就好**

### 4. 两步走：先分析，再拆分

这工具是**辅助性质**的，设计上就是两步：

```
分析 → 人工确认 → 拆分
```

分析完**一定一定**要把表格拉一遍，人工确认没问题了再点拆分。目测能省掉**70%**一页页拆+重命名这种蠢活，剩下**30%**还是得靠人。

### 5. 配图号标签

图框里的图号字段叫法不统一，有的叫"图号"，有的叫"图纸编号"，有的叫"编号"……**同一份图里是一致的**，但不同项目可能不一样。打开PDF看一眼图框，输入对应的标签名就行。**空格会被自动忽略**。
